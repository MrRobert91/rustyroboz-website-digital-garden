An accessibility web app that lets people with severe mobility impairments fill in Google Forms and Microsoft Forms using only eye-gaze control.

It uses webcam eye tracking with MediaPipe, guided calibration, dwell-based Yes/No selection and text-to-speech. The frontend is React + Vite and the backend is FastAPI + SQLite + Piper.

A personal context layer for AI agents that turns scattered notes (Obsidian vaults, project repos) into actionable context. It stays read-only over your notes, with any AI output isolated in an `_ai/` folder.

The system exposes an MCP server for agent queries, CLI pipelines for indexing and retrieval, and a React dashboard. The backend is built with FastAPI, FastMCP and pydantic-ai.
